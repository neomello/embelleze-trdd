## ⟠ Rotina de Operação

1. Identifique a seção afetada
2. Edite apenas o arquivo necessário
3. Dado de negócio → prefira `src/content/`
4. CTA WhatsApp → use `src/lib/whatsapp.ts`
5. Tracking → use `src/lib/tracking.ts`
6. Preserve `index.astro` limpo

────────────────────────────────────────

## ⍟ Contrato de Seção

Cada seção deve:

```text
▓▓▓ REQUISITOS POR SEÇÃO
────────────────────────────────────────
└─ Ter uma função clara de conversão
└─ Ter CTA quando fizer sentido
└─ Ser legível no mobile
└─ Não depender de outra seção
   para funcionar
```
